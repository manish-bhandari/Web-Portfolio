Projects

Here are some of the projects I have worked on recently. Hover over each item to learn more!

Laundry Line

iOS and Android application using React Native and Firebase that helps students overcome issues they face with the laundry system in dormitories. Users can update the realtime statuses of washers and dryers. User authentication and texting communication features also included.

UT Course Notifier

Created a web scraping tool that automatically keeps track of a course's open or close status and notifies the user of any changes. Written in Selenium and AutoHotKey.

Horns Race

A speed typing game created using React where uses can test their typing abilities including speed and accuracy.

Wolf Connect

An iOS mobile application created using XCode and Swift for Westside high school students. Wolf Connect is a platform where students can access several school resources and get daily announcement notifications.

Covid Signals

A web application that marks the threat of COVID-19 via tweets from across the country. Used twiter's TweetyAPI to analyze tweet messages and locations.
